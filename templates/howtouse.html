How to use the Med analyser

This software is made keeping in mind the doctors and to help them analyse the patient history, prescriptions and other records

Here you’ll know how to use this software

Getting some sample data

There is some data available for you to test, the agent can take pictures and pdf as inputs and you can ask question to it.

Click on the Sample files button on the right header

Then you’ll see some available medical records photos you can download

You can preview it and download each file by clicking the download button

Uploading the data

Now that you have the sample data, drag it to the Upload Medical Documents or just select it from the downloads folder

Note: Don’t upload more than 16 MB of files, else it will be rejected

Now press upload, and wait for it to upload, this may take some while so be on the page.

Using the Agent

Now that you have given the agent your data to ask question, you can just type it out in the input and press enter

The agent will give you answer in accordance with the file that you have uploaded.

And it will also reference the file from where it was taken from

the yellow highlight tells you which document is the reference from.

There you go, now you can ask what you wish to related to any medical data.
